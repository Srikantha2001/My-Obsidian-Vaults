## INTRODUCTION

- The application is a simple Online Book Store that involve following microservices.
	- <mark style="background: #FFB8EBA6;">CATALOG_SERVICE</mark> : For fetching the books
	- <mark style="background: #FFB86CA6;">ORDER_SERVICE</mark> : For handling orders of books
	- <mark style="background: #ADCCFFA6;">NOTIFICATION_SERVICE</mark> : For Notifying users regarding the status of Order
- Following Tech-stack is used for development 
	1. Spring Boot - backend
	2. JPA (Postgres) with Flyway - database
	3. Docker - Image generation and running application
	4. Prometheus - Monitoring
	5. SDKMAN - For Managing java version for application
